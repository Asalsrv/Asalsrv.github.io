## Files

- `index.html` - homepage
- `resume.html` - CV page
- `roadmap.html` - research focus page
- `blog.html` - research notes page
- `why-this-site.html` - first note
- `contact.html` - contact page
- `resume-blog.css` - complete styling
- `art-dna.png` - DNA illustration used near genomics and research sections
- `art-lab-glassware.png` - laboratory glassware illustration used near chemistry sections
- `art-computer.png` - computer illustration used inside Computer Skills
- `art-star-pink.png`, `art-star-yellow.png`, `art-star-blue.png` - decorative star images placed in text-free quadrilateral frames
- `art-yellow-blob.png`, `art-pink-blob.png` - decorative abstract images
